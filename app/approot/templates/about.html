About This Demo

This demo deploys three mongo database servers and configures them in a replicaset for redundancy. It also deploys a Flask based python web application that uses the mongo database servers to deliver a view of the mongo data. The application is deployed using a single CloudScript template which orchestrates the creation of all resources or even missing resources.

Deployment

The applicaton uses first boot scripts on NephoScale base images to create servers that do useful work from basic images. Each of the first boot scripts downloads a github hosted application and depending on the role of the server, runs the appropriate setup scripts.

Idempotency

The demo utilizes NephoScale CloudScript and idempotency to create missing resources (servers) which are missing and need to be created from scratch. This means that we can make a simple change on github, delete our mongoapp server and re-run the CloudScript document. The NephoScale platform will intelligently create the missing resource using the latest github commits.
